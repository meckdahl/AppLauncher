üîß Troubleshooting & Debug Info

Most of the time, everything works perfectly! But if you're having issues, this section will help.

Mac/Linux: If Double-Click Doesn't Work

Sometimes Mac or Linux needs an extra step the first time:

  1. Open "Terminal" (search for it in your applications)
  2. Type: cd then press Space
  3. Drag the folder containing the files into the Terminal window
  4. Press Enter
  5. Type: chmod +x MAC_DoubleClick_to_Start.sh
  6. Press Enter
  7. Type: ./MAC_DoubleClick_to_Start.sh
  8. Press Enter

After this first time, double-clicking will work!

Windows: "Python is not installed" Message

If you see this message, the installer will try to download Python automatically. If that fails:

  1. Go to: https://www.python.org/downloads/
  2. Download Python (latest version)
  3. IMPORTANT: During installation, check the box "Add Python to PATH"
  4. Complete the installation
  5. Restart your computer
  6. Double-click the WIN file again

Linux: "tkinter is not available" Message

If you see an error about tkinter, install it using your package manager:

Ubuntu/Debian:

sudo apt-get update sudo apt-get install python3-tk

Fedora:

sudo dnf install python3-tkinter

Arch:

sudo pacman -S tk

After installing, run the MAC file again.

"UV installation failed" or "Fast installer" Error

If the automatic UV installation fails:

  1. Check your internet connection
  2. Try running the file again (it might have been a temporary issue)
  3. If it still fails, the app will work anyway - it just might be a bit slower

The App Window Never Opens

If the setup completes but the app doesn't open:

  1. Look for any error messages in the window
  2. Try running the file again
  3. Check if there's a file called launcher.py in the folder
  4. If launcher.py is missing, you may have an incomplete download - download the files again

Manual Installation (Advanced)

If automatic setup isn't working, you can install things manually:

1. Install Python:

  • Download from python.org
  • Install Python 3.8 or newer
  • Make sure "Add to PATH" is checked during installation

2. Install UV:

pip install uv

3. Run the launcher manually:

python launcher.py

What Happens When You Double-Click?

Understanding what the file does might help with troubleshooting:

  1. Checks for Python - Looks for Python 3.8 or newer
  2. Installs Python if needed - Downloads and installs the latest version
  3. Installs UV - A fast package installer for Python apps
  4. Creates folders - Sets up the apps folder structure
  5. Launches the app - Starts launcher.py
